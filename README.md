# react-native-fun-loadings

Traditional and fun loadings for you to use in your projects!

## Installation

```sh
npm install react-native-fun-loadings
```

## Usage


```js
import { multiply } from 'react-native-fun-loadings';

// ...

const result = multiply(3, 7);
```
